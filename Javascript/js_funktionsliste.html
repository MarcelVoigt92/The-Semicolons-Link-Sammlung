javascript-builtin_functions:



Functions are one of the fundamental building blocks in JavaScript. A function in JavaScript is similar to a procedure—a set of statements that performs a task or calculates a value, but for a procedure to qualify as a function, it should take some input and return an output where there is some obvious relationship between the input and the output. To use a function, you must define it somewhere in the scope from which you wish to call it.


Function declarations


A function definition (also called a function declaration, or function statement) consists of the function keyword, followed by:

-- The name of the function.

-- A list of parameters to the function, enclosed in parentheses and separated by commas.

-- The JavaScript statements that define the function, enclosed in curly brackets, { /* … */ }.



Funktionen sind die grundlegenden Bausteine ​​in JavaScript. Eine Funktion in JavaScript ähnelt einer Prozedur – einer Reihe von Anweisungen, die eine Aufgabe ausführen oder einen Wert berechnen. Damit eine Prozedur als Funktion qualifiziert werden kann, sollte sie einige Eingaben annehmen und eine Ausgabe zurückgeben, bei der eine offensichtliche Beziehung zwischen ihnen besteht ==> Eingang und Ausgang. Um eine Funktion zu verwenden, müssen Sie sie irgendwo in dem Gültigkeitsbereich definieren, von dem aus Sie sie aufrufen möchten.


Funktionsdeklarationen


Eine Funktionsdefinition (auch Funktionsdeklaration oder Funktionsanweisung genannt) besteht aus dem Schlüsselwort function, gefolgt von:

-- Der Name der Funktion.

-- Eine Liste von Parametern für die Funktion, die in Klammern eingeschlossen und durch Kommas getrennt sind

-- Die JavaScript-Anweisungen, die die Funktion definieren, eingeschlossen in geschweiften Klammern, { /* … */ }.

Die toExponential() - Function

Forces a number to display in exponential notation, even if the number is in the range in which JavaScript normally uses standard notation.

Erzwingt die Anzeige einer Zahl in Exponentialschreibweise, selbst wenn die Zahl in dem Bereich liegt, in dem JavaScript normalerweise die Standardschreibweise verwendet.


Beispiel:

let totn_number = 123.456789; console.log(totn_number.toExponential()); // ===> 1.23456789e+2 console.log(totn_number.toExponential(1)); // ===> 1.2e+2 console.log(totn_number.toExponential(2)); // ===> 1.23e+2

Die toFixed() - Function

Formats a number with a specific number of digits to the right of the decimal.

Formatiert eine Zahl mit einer bestimmten Anzahl von Ziffern rechts vom Dezimalkomma.


Beispiel:

let totn_number = 123.456789; console.log(totn_number.toFixed());
// ===> 123
console.log(totn_number.toFixed(1));
// ===> 123.5
console.log(totn_number.toFixed(2));
// ===> 123.46

Returns a string value version of the current number in a format that may vary according to a browser's locale settings.

Gibt eine Zeichenfolgenwertversion der aktuellen Zahl in einem Format zurück, das je nach den Gebietsschemaeinstellungen eines Browsers variieren kann.


Beispiel:

let totn_number = 123456.789; console.log(totn_number.toLocaleString());
// ===> 123,456.789

Die toPrecision() - Function

Defines how many total digits (including digits to the left and right of the decimal) to display of a number.

Legt fest, wie viele Stellen insgesamt (einschließlich Stellen links und rechts vom Dezimalkomma) einer Zahl angezeigt werden.


Beispiel:

function precise(x) { return x.toPrecision(4); } console.log(precise(123.456));
// ===> "123.5"
console.log(precise(0.004));
// ===> "0.004000"
console.log(precise(1.23e5));
// ===> "1.230e+5"
